# Матрица CSIR

Код генерирует положительно определённую симметричную матрицу в формате CSIR, случайный вектор правой части и решает систему уравнений методом сопряженных градиентов.

Метод сопряжённых градиентов может быть реализован в случае, если матрица `A` является положительно определённой матрицей, следовательно и симметричной. Для хранения матрицы при реализации метода сопряженных градиентов был использован метод CSIR. Он хорошо подходит для хранения разреженных матриц, значащие элементы которых (ненулевые) сконцентрированы рядом с главной диагональю (в частности, ленточные матрицы).

Суть метода CSIR заключается в хранении матрицы `n × n` (поскольку метод сопряжённых градиентов предназначен для решения СЛАУ с квадратными матрицами в силу их симметричности, реализация заточена под оперирование квадратными матрицами) с помощью четырёх векторов:
`di` размерности `n`, в котором хранятся диагонали (всегда ненулевые для положительно определённых матриц, что делает элементы вектора всегда значащими в контексте задачи);
вектора `elem`, в котором хранятся построчно недиагональные элементы, начиная с первого значащего (ненулевого) для строки;
вектора `ja` размерности `n`, в котором содержится колонка первого ненулевого элемента каждой строки;
вектора `ia` размерности `n + 1`, в котором хранятся индексы начала и конца (не включительно) строки внутри вектора `elem`.

Такая структура хранения позволяет для симметричных матриц хранить только половину значений недиагональных элементов, что дополнительно сокращает расходы на хранение матрицы.

Генерация матрицы производится на основе теоремы:
«Матрица `A` является положительно определенной тогда и только тогда, когда существует разложение
`A = LLᵀ`, называемое разложением Холецкого, где `L` – нижняя треугольная матрица с ненулевыми элементами на диагонали».
